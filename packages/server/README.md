# BACKEND

built with Node.js, Express and GraphQL.

* ORM: Sequilize
* DB: Postgres


## Usage


### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve the beckend

```sh
$ npm start
```

To configure the DB go to src/models/index.js

To fill the database

```sh
$ npm run initDb
```

### Build

```sh
$ npm run build
```

## Running unit tests

```sh
$ npm run test
```
